Paste rows copied from Excel/Sheets or a CSV. One product per line, columns in this order:
Category, Name, Case Cost, # per Case, Unit Cost, Sell Price, Link
Case Cost, # per Case, Unit Cost, and Link may all be left blank individually — but if Case Cost and # per Case are both filled in, Unit Cost is calculated automatically (plus a $0.10 deposit for Beverage items) and any value you paste there is ignored. Works with commas or tabs. Older 5 or 6-column lists still work fine. A header row is skipped automatically.
